## 🚀 Featured Projects

### 🫁 [Explainable Lung Cancer Detection System](https://github.com/preyanshnahar/Lung-Cancer-Detection-using-Deep-Learning)
> TensorFlow · Keras · Xception CNN · SHAP · Grad-CAM · Streamlit

Xception-based CNN using transfer learning to classify lung CT scans into 4 cancer types. Achieved **99.18% training** and **88.89% validation** accuracy. Features Explainable AI (Grad-CAM + SHAP) and a Streamlit web app for CT upload and heatmap visualization.

---

### 🧬 [Non-Hodgkin Lymphoma Classification](https://github.com/preyanshnahar/AN-EXPLAINABLE-DEEP-NEURAL-NETWORK-APPROACH-FOR-NON-HODGKIN-LYMPHOMA-CLASSIFICATION)
> Deep Learning · Grad-CAM · SHAP · Jupyter Notebook

An explainable deep learning framework for classifying Non-Hodgkin Lymphoma subtypes (MCL, FL, CLL) from histopathological images. Integrates Grad-CAM and SHAP to visualize model decisions and improve clinical interpretability.

### 🔐 [Leak_Hunter — Memory Leak Detector in C](https://github.com/preyanshnahar/Memory-Leak)
> C Language · Preprocessor Macros · atexit

Lightweight memory leak detector for C projects. Automates tracking using preprocessor macros for malloc/free and generates detailed leak reports on program exit via `atexit(leak_report)`.
